## abc-ide:n käyttö-ohjeet	

abc-ide on yksinkertainen koodi editori. Voit koodata ja ajaa Java ohjelmia. Painamalla "Run" nappulaa koodi käännetään ja ajetaan. Ajon aikana ohjelman output näkyy ohjelman alalaidassa olevassa "konsolisa". Konsolin alla on myös input tekstikenttä, millä voit antaa inputtia ohjelmaan (mikäli ohjelmasi kysyy esim Scannerin avulla käyttäjältä inputtia).

Save nappula tallentaa koodin valitsemaasi tiedostoon, Load nappula lataa valitsemasi tiedoston ja New nappula resetoi koodin Hello Worldiksi.
